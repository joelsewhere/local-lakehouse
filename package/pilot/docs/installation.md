# Installation

## Stable release

To install pilot, run this command in your terminal:

```sh
uv add pilot
```

Or if you prefer to use `pip`:

```sh
pip install pilot
```

## From source

The source files for pilot can be downloaded from the [Github repo](https://github.com/iamnopilot/pilot).

You can either clone the public repository:

```sh
git clone git://github.com/iamnopilot/pilot
```

Or download the [tarball](https://github.com/iamnopilot/pilot/tarball/master):

```sh
curl -OJL https://github.com/iamnopilot/pilot/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd pilot
uv pip install .
```
